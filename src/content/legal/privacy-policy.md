本站是一个纯静态的个人分享网站，**不收集、不上传、不追踪任何用户个人数据**。我们没有服务器数据库，也没有任何后台在记录你是谁、看了什么。

## 一、我们不做这些

- **不收集个人信息**：本站没有注册、登录、评论、留言或表单功能，不会向你索取、也无法获取你的姓名、邮箱、手机号等任何个人信息。
- **不做用户追踪**：本站**没有接入任何统计分析或广告系统**（如 Google Analytics、百度统计、广告联盟等），不会给你建立画像、不投放个性化广告。
- **不设追踪 Cookie**：本站自身不写入任何用于识别或追踪你的 Cookie。
- **数据不出你的设备**：本站没有服务器端存储，你的任何操作都不会被上传到我们这里。

## 二、只保存在你本机的偏好数据

为了让你用得顺手，本站会用浏览器的**本地存储（localStorage）**记住几项设置。这些数据**只保存在你自己的这台设备上，永远不会上传到任何服务器**，我们也无从读取：

- **主题偏好**：记住你选择的深色 / 浅色模式。
- **我的收藏**：你在「收藏」里保存的景点、电影、剧集清单。
- **打卡记录**：你在「打卡」页勾选的已去 / 已看条目。

你随时可以清除它们：在浏览器里清除本站的网站数据，或直接在「打卡」页点「清空全部打卡」，相关记录即被彻底删除，不留任何副本。

## 三、托管方与访问日志

本站托管在 **GitHub Pages** 上。当你访问时，托管方（GitHub）可能会按其自身隐私政策记录标准的服务器访问日志（如 IP 地址、浏览器类型），用于安全与运营。**这部分由 GitHub 掌握，本站无法访问、也不参与其中。** 详情可参阅 GitHub 的隐私声明。

## 四、第三方嵌入内容

文章中会嵌入来自第三方平台的内容，主要是：

- **视频**：来自 YouTube / Bilibili 等平台的官方预告片（以内嵌播放器形式）。
- **图片**：影视海报、剧照等静态图片已尽量自托管在本站。

当你播放这些第三方视频时，对方平台可能会按其自己的政策设置 Cookie 或收集数据。这部分不受本站控制，请参阅相应平台各自的隐私政策。如果你不希望被这些平台记录，可以选择不点击播放。

## 五、面向未成年人

本站为一般性的观影与旅行推荐内容，不面向未满 14 周岁的儿童定向收集任何信息。

## 六、政策更新

本政策可能会不时更新，最新版本以本页为准，页首标注「最后更新」日期。更新后继续使用本站即视为知悉修订后的内容。

## 七、联系

如对本隐私政策有任何疑问，可在[关于页](/about)了解更多。

**一句话总结：本站不要你的账号、不记录你的身份、不追踪你的行为；收藏和打卡只存在你自己的浏览器里。**
